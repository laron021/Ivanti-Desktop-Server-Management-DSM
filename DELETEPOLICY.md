# FORCEREMOVE

## DESCRIPTION
+ Azonnal törli a megadott policy-t uninstall nélkül

## SYNTAX

```ruby
forceremove [[-ID] <Int32>] 
```

## EXAMPLES

```powershell
forceremove -ID 12215 

```

## PARAMETERS

### -ID

```yaml
Type: Int32
Mandatory: True
```


# REMOVE

## DESCRIPTION
+ Uninstallálja a megadott policy-t

## SYNTAX

```ruby
remove [[-ID] <Int32>] 
```

## EXAMPLES

```powershell
remove -ID 123387

```

## PARAMETERS

### -ID

```yaml
Type: Int32
Mandatory: True
```
