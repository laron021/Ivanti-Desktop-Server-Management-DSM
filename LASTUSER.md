# LASTUSER

## DESCRIPTION
+ Megadja, hogy egy adott user melyik gépen volt utoljára bejelentkezve

## SYNTAX

```ruby
lastuser [[-username] <String>] 
```

## EXAMPLES

```powershell
lastuser -username 'ABCD17'

```

## PARAMETERS

### -username
DSM group object ID

```yaml
Type: String
Mandatory: True
```
